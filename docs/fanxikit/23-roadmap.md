# 23. Roadmap triển khai

## Phase 1

- File-based routing.
- Route manifest.
- Basic SSR skeleton.
- Basic client navigation.
- Basic `fanxi-adapter-node` output.

## Phase 2

- Layouts.
- Load functions.
- Server endpoints.
- Form actions.
- Error pages.

## Phase 3

- Hydration orchestration.
- Data serialization.
- Head/CSS collection.
- HMR.
- Type generation.

## Phase 4

- Prerender.
- `fanxi-adapter-static`.
- `fanxi-adapter-cloudflare`.
- SPA mode.
- CSR/SSR per-route config hoàn chỉnh.

## Phase 5

- Preloading nâng cao.
- Invalidation tối ưu.
- Streaming SSR.
- Advanced adapters.
- Devtools integration.
