# 01 - Tổng quan fanxipan

fanxipan là framework frontend compiler-first, dùng Rust compiler để biến `.fanxi` thành mã JavaScript cập nhật DOM trực tiếp.

## Triết lý

- Compile-time first.
- Runtime nhỏ và ổn định.
- Fine-grained reactivity.
- Không virtual DOM.

## Mục tiêu sản phẩm

- DX nhanh như các compiler framework hiện đại, nhưng syntax công khai là Fanxi function component riêng.
- Hỗ trợ SSR, hydration, router, HMR, plugin.
- Kiến trúc rõ ràng cho mở rộng lâu dài.
